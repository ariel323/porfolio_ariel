🔧 Instrucciones de Uso

1. Descargar las imágenes:

docker pull ariel234/caso-001-app:latest
docker pull ariel234/caso-001-mysql:latest

2. Ejecutar MySQL:

docker run -d --name mysql-db ariel234/caso-001-mysql:latest

3. Ejecutar la aplicación:

docker run -d --link mysql-db --name app ariel234/caso-001-app:latest
